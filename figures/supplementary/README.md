# Supplementary figures

The four supplementary figures shown in the paper are:

| File                | Description                                         | Generated by                     |
| ------------------- | --------------------------------------------------- | -------------------------------- |
| `figureS1.jpeg`      | GeoSOT-3D global frame & 2-degree quadtree concept  | static; see `static/figureS1.svg` |
| `figureS2.jpeg`      | Synthetic-scene layout (terrain + buildings + zones)| `make_figS2.py`                  |
| `figureS3.jpeg`      | 3D terrain visualisation of the Xi'an study area    | `make_figS3.py`                  |
| `figureS4.jpeg`      | Local detail comparison (Uniform vs TSDP)           | `make_figS4.py`                  |

Run any of the regenerators with:

```bash
python -m figures.supplementary.make_figS3 \
    --input results/xian_partitions/tsdp.npz \
    --output figures/output/figureS3.jpeg
```

Each script exposes `--input` and `--output` flags; defaults match the layout
expected after running `experiments/xian.py`.

## Static SI figure (S1)

The conceptual schematic in S1 is hand-drawn in `figures/static/figureS1.svg`
and exported via Inkscape / rsvg-convert. It is not generated from data.
